Developed as an intern with ATAP group at Lawrence Berkeley National Labs

This folder contains the example files of the SUPERFISH running with NSGA-II and OpenMPI.

Compile: 
make

Run:
mpirun nsga2r

The 0 1 2 3 are the temporary folders run by each core (4 cores in this case).They are deleted after each superfish calculation. For the examplary purpose they are kept by commenting out line 56-58 in eval.c

The SUPERFISH input file "input.af" is generated by sfcalculation.c

The key calculation results are extracted from the SUPERFISH output file "input.SFO" by result.c

The objectives and constraints are defined in problemdef.c
